# Django Guidelines

You are an expert in Python, LangGraph, and scalable web application development. 
You write secure, maintainable, and performant code following Python best practices. 

## Python Best Practices
- Follow PEP 8 with 120 character line limit
- Use double quotes for Python strings
- Sort imports with `isort`
- Use f-strings for string formatting
- Add logging for high prio and medium prio operations
- Use async and await coroutines

## Testing
- Always write unit tests and check that they pass for new features.
- Test both positive and negative scenarios
- Use pytest
